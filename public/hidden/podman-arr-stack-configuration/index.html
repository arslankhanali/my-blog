Arr Stack - Configuration

August 8, 2025
#arr-stack
#podman
#homelab

Introduction

The “Arr stack” is a popular set of applications for automating your media library. This blog post will walk you through configuring a basic Arr stack using Podman on your homelab. We’ll cover Jellyfin for media streaming, Jellyserr for integrating services, Prowlarr for managing indexers, and Sonarr and Radarr for automating TV and movie downloads. This setup provides a simple yet powerful way to manage your media. 🚀

Configuring the Arr Stack

Basic flow about how each service talks to each other

  1. Jellyfin
    1. Add media and scan
      1. Enable trickplay image extraction
      2. Save trickplay images next to media
    2. Trickplay Image Interval = 300000
      1. Only generate images from key frames
  2. Jellyserr
    1. Settings -> Jellyfin. Get API from Jellyfin (Advanced -> API Keys) - See pic js1
    2. Settings -> Services -> Radarr. Get API from Radarr (Settings -> General)
    3. Settings -> Services -> Sonarr. Get API from Sonarr (Settings -> General)
  3. Make sure flare-bypasser container is up
  4. Prowler
    1. Add indexers -> YTS etc
    2. Settings -> Indexer -> FlareSolverr -> Setup flare-bypasser
    3. Settings -> Apps -> Applications -> Radarr
    4. Settings -> Apps -> Applications -> Sonarr
  5. Sonarr
    1. Select location for TvShows
    2. Settings -> Indexer -> Verify indexers (from Prowler)
    3. Settings -> Download Clients -> Transmission
  6. Radarr
    1. Select location for Movies
    2. Settings -> Indexer -> Verify indexers (from Prowler)
    3. Settings -> Download Clients -> Transmission

Summary

Congratulations! 🎉 You’ve now configured a powerful and automated media stack. With this setup, you can add movies and TV shows from your phone or desktop, and the Arr stack will automatically find and download them, ready for you to watch on Jellyfin. This is a great way to streamline your homelab and take control of your media.
